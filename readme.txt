== Description ==

Description: This plugin allows you to implement the advanced system of statistics of logonike.com. It will help you understand the development of your website. It gives you information you will not find in other similar services. It will help in the positioning of your website. A simple code that collects information and processes it for positive purposes. Get the id of your website and of user and start enjoying the benefits of having your system statistics on a remote server.

= Features =

- You can see visitors to your websites in real time.
- You can have the report of your visitors every hour, every day, every month and more.
- High-quality graphics.
- Completely intuitive.
- Unique visitors and all visits in general.
- Know where your visitors come. Scans all the search engines.
- Know what are the most visited pages of your website.
- Know what words are positioned on search engines.
- Know of what countries your visitors are.
- And more...

== Installation ==

Statistics Website is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

1. Copy el plugin from the downloaded and extracted file.

2. Paste it in wp-Content/plugins folder on your Wordpress Installation 

3. Activate the plugin from Dashboard / Plugins window.

4. Now Plugin is Activated.
